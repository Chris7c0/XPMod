Vanilla L4D2 default health values are used as the base. The standard L4D2 values are:
  - Smoker: 250 HP
  - Boomer: 50 HP
  - Hunter: 250 HP
  - Spitter: 100 HP
  - Jockey: 325 HP
  - Charger: 600 HP
  - Witch: 1000 HP
  - Tank: z_tank_health default is 4000 (×1.5 = 6000)
